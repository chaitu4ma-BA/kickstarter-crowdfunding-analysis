
# MS Excel Analysis

This folder contains Microsoft Excel files used for data preparation, analysis, and reporting in the Kickstarter crowdfunding project.

Key activities performed in Excel:
- Data cleaning and formatting
- Handling missing and inconsistent values
- Pivot tables for summary analysis
- Lookup functions (VLOOKUP / XLOOKUP)
- Calculated fields and KPIs
- Trend and comparative analysis

Excel deliverables include:
- Raw and cleaned datasets
- Analysis workbooks
- Excel-based dashboards and summary reports

Excel was used as the primary tool for initial exploration and validation of insights before dashboard creation.
