# Dashboards

This folder contains dashboards and visual reports created to present insights from the Kickstarter crowdfunding analysis.

Tools used:
- Excel
- Power BI
- Tableau

Key insights covered:
- Overall campaign success and failure rates
- Category-wise funding performance
- Goal amount vs pledged amount comparison
- Time-based trends in campaign launches and funding

Dashboard files include:
- Excel dashboards and summary reports
- Power BI (.pbix) interactive dashboards
- Tableau (.twbx) interactive dashboards

Screenshots are provided for quick viewing without opening dashboard files.
